Your Feedback Analytics account includes a tool to create custom themes. These custom themes can be applied to your forms.
To create a custom design, you'll need a free Feedback Analytics account. Here is how to set one up.
Log in to your account, and you'll land on the dashboard. Navigate to the Design tab in the top menu.
The left side of the page contains out of a list with all the customisable components. All changes in the design need to be manually saved with the save design button below.
Always save the design by clicking the save design button.
On the right side of the page there are two example cards. These show the selected design and all changes in real time.
To select another design, you click on Name. After clicking, On the bottom of the page a drop-down menu appears. To select another design simply open this menu and select the preferred one. If you haven't created any designs yourself, this list will be empty.
To create a new design click on the + icon below the dropdown menu. A pop-up window will appear where you can give your custom design a name.
To delete an existing design, select the page you want to delete in the dropdown menu. Now click on the - icon below the dropdown menu. A Confirmation message will appear. Click on delete to proceed.
To copy an existing design, select the page you want to copy in the dropdown menu. Now click on the copy icon below the dropdown menu. A pop-up window will appear where you can give your copied design a name.
To get a design back to default values, select the page you want to refresh in the dropdown menu. Now click on the refresh icon below the dropdown menu. A pop-up window will appear where you can confirm the refresh. After clicking this the design will be back to its default values.
To change the name of an existing design start typing in the name tab. Press the enter key or press somewhere on the screen to confirm the changes.
To change the font type click on font. Now open the dropdown menu and select the preferred font.
The color of the fonts can be changed. Questions is the title of each question card and also the title of the start and end screen. Also the fontcolor of the button will change. These buttons are only on the open questions, start screen and end screen.
The font color of the provided Answers can be changed, this will also be the color of the extra text provided with some questions.
The color of the Next Buttons, question Cards and Background can be customised.
Selecting a color can be done manually or you can fill in a HSL, Hex or RGB value.
