It's possible to add multiple profiles to your Feedback Analytics account. Each profile with their own login credentials. The amount of profiles that can be active at the same time is dependent on your subscription.
Log in to your Feedback Analytics account and navigate to the settings page. Click here if you dont have a account yet.
Scroll down to the Users section. This is the place where all profiles are managed. Some of the possibilities are adding new profiles, changing user rights, and deleting existing profiles.
To edit a profile click on the pen symbol on the right. It's possible to change the first name, last name and the profile rights. The different profile rights are admin, readwrite, readonly and none.
None means that a user has zero rights. This person isn't allowed to log in. Ideal if you have more colleagues as allowed active spots in your Feedback analytics account. If you have 5 colleagues, but are limited to 3 active accounts, 2 accounts should be put on none. If you want to give another person editing rights, you should first put someone else on none, before giving this person extra rights.
Read has the rights and is limited to reading and testing forms.
Readwrite has the rights to read and edit forms. However this person can not publish forms.
The admin can do everything. Publishing forms and also editing user rights. Even deleting other admins is added to the toolkit.
To delete a user click on the red thrash can on the right. A confirmation window will appear, press the delete button to proceed.
To add a new user click on the green + symbol. A pop up window will appear. Fill in de details and click on done. The user is automatically added to the list. Most of the times the new user can log in immediately, but it's possible this process takes up to 24 hours. If logging in is still not possible after 24 hours please contact the support center.
