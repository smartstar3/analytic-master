Boolean logic is a form of algebra that is centered around three simple words: "Or", "And" and "Not". A basic understanding of this logic, makes using the logic jumps a lot easier.
A boolean statement is always either true or false, there is no in between. With logic jumps we want a question to jump to a certain question if the boolean statement returns true, and to another question if it returns false.
"OR" is used to express that as long as one of the multiple conditions is met, the value of the boolean statement is true.
Example: If we have either apples OR cherries we could make a pie.
"AND" is used to indicate that all the multiple conditions must be met. If all the conditions are true, our boolean statement returns true, else it returns false.
Example: If I have a computer AND an internet connection then I can do some work.
"NOT" is used to show that a certain condition should be excluded. If this certain condition is excluded, the boolean statement returns true, else it returns false.
Example: If you're NOT younger than 16, you can watch this movie.
