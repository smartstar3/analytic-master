Logic jumps is a powerful tool which can be quite overwhelming when just starting out. We are going to introduce you to this topic using a few simple example.
example 1
To add Logic jumps you should now how to create a form with Feedback Analytics. If you dont know yet how to do this, read this first.
Create a new form and add 3 questions to it. A yes/no, an open and another yes/no -question respectively.
If someone fills in that they like dogs, the form should go to the second question. However, if they dont like it, the form should jump directly to the 3rd question. Go to the first question in the builder and click Add Logic Jump.
If the first answer is a yes, we want to go from question 1 to 2. Questions always automatically go to the next question on the list, so we dont need to change anything for this behaviour. When the answer is no, the form should jump to question 3. So we change the logic to "IF Do you like dogs? is NO, THEN "Do you like cats?"
example 2
We are going to create a form with a multiple choice question. For each of the possible choices, the form jumps to another end screen. If you don't know how to create different end screens, read this short tutorial.
Create a form with a single Multiple Choice question
Now create 3 different end screens. For each selected answer we are going to a different end screen.
Click Add Logic Jump and fill in the following values "if the multiple choice IS choice 1 THEN go to end screen 1"
Click on the green + below the logic jump and do the same for end screen 2 and 3.
example 3
We are going to create a form which looks at multiple answers to decide on a jump. Create a form with 2 yes/no questions and a single rating question respectively.
Click on the latest question that has a input on the jump decision. In this case that's the second yes/no question. Subsequently add a logic jump.
Without any logic the form always goes to the rating question, however in this case we dont want the rating if the customer was negative about any of the experiences.
This translates to IF question 1 is negative OR IF question 2 is negative THEN go to the end screen.
