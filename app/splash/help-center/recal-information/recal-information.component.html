Feedback Analytics forms are responsive. This means that information gathered in a previous question, can be used in another. If this is not fully clear yet, there will be an example in this document.
To add recall information you should now how to create a form with Feedback Analytics. If you dont know yet how to do this, This place is where you should be.
Recall information can be used on Open questions, Multiple Choice questions, and number questions. In this document we will set up a simple example with all of them
We have created a form with a Open question and a Multiple Choice question
For this example we add a new number question to the form, However this could be any other question type.
We want to know the quantity of fruit that has been bought, but at the same time we want to have a interactive form.
We want the question to be in the following structure: how many [fruit] did [name] buy?
Have the cursor on the place you want the recalled information and click on the + icon
A dropdown window pops up. Select the question where you want the answer from. In this case we want question 2, the fruit.
Afterwards we do the same for the name.
You can make this as easy or complicated as you want. When you combine recall information with logic jumps, keep in mind that recalled questions need to be answered to appear, else they will leave a blank.
